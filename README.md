# 📊 DataDigger SQL Project

## 📌 Overview
This project is a structured SQL-based database system designed to manage and analyze customer, order, and product data. It demonstrates real-world database operations including table creation, data manipulation, and analytical queries.

## 🚀 Features
- Creation of relational database tables (Customers, Orders, Products, OrderDetails)
- Implementation of CRUD operations (Create, Read, Update, Delete)
- Data filtering using WHERE conditions
- Sorting and organizing data using ORDER BY
- Aggregate functions (SUM, AVG, MAX, MIN)
- Real-world business query simulations

## 🛠️ Technologies Used
- SQL (MySQL)
- Database Management System (DBMS)

## 📂 Database Structure
- **Customers** – Stores customer details
- **Orders** – Contains order information
- **Products** – Stores product details
- **OrderDetails** – Maintains order-wise product transactions

## 📊 Key Operations Performed
- Data insertion and updates
- Deleting records based on conditions
- Retrieving specific data using filters
- Performing revenue calculations
- Finding highest and lowest values

## 🎯 Purpose
The goal of this project is to build a strong foundation in SQL and understand how databases are used in real-world business scenarios like e-commerce and inventory management.

## 📌 Conclusion
This project showcases practical SQL skills and demonstrates how structured data can be managed, analyzed, and optimized for decision-making.
